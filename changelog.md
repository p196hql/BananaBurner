## Script v3.0
## Extension v2979.0.9

### BananaBurner 2979 Script v3.0 is out!
- Ability to view and manage shared servers:
  - can be added to quick actions
  - fixed and improved permission checks
  - plugin parity for shared server cards
- Templates for python servers:
  - basic Flask template
  - basic pycord discord bot template
- VFS cache for faster directory browsing and file/folder actions
- Telemetry is setup and if you have "Share Telemetry" enabled in settings, data that helps improve BananaBurner 2979 will be sent to Termux Labs.
  - current coins, plugins installed, theme settings, configuration and other details are sent, credentials and personally identifiable information is not collected
     - to see what exactly gets sent, click "Inspect Popup" on the extension  in   your browser extension list to open devtools console, at the top, browser over to "Network" wait for over 60 seconds for a request to appear called "telemetry" click on it and head over to the payload tab to see the raw payload that gets sent.
- Bug fixes and improvements
- Google Translate fixes
- Other

Update by reinstalling the extension.
